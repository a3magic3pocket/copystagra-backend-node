# copystagra-backend-node
인스타그램을 모방한 [copystagram](https://sonminsoo.net)의 node.js backend 소스코드입니다.

## 연관 프로그램
- kafka:3.7.0
- mongodb:jammy

## 의존성 패키지 설치
- ```bash
  npm ci
  ```

## 개발버전 실행
- ```bash
  npm run start:dev
  ```

## 상용버전 실행
- ```bash
  npm run start
  ```

## 테스트
- ```bash
  npm run test
  ```
